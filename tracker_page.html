About these numbers

Money raised / spending / cash on hand come from each committee's processed FEC reports, which update nightly and lag real activity by about a day.

Real-time IE estimate sums independent-expenditure line items from the FEC's real-time e-filing feed (last ~4 months). It is an unofficial leading indicator — not amendment-resolved — so the processed independent-expenditures figure is authoritative.

Electioneering applies to filers like Public First Action Inc. that report candidate-naming issue ads (Form 9) rather than express-advocacy IEs. Such groups don't itemize donors to the FEC, so their raised figure is intentionally blank.
